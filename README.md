# বয়ন - সার্বভৌম নেটিভ অ্যাসেম্বলার

**বয়ন** হলো নির্দেশিকা-১৬ (Nirdeshika-16) জন্ত্র-বিধানের জন্য তৈরি একটি মৌলিক এবং স্বাধীন অ্যাসেম্বলার। এটি 'অক্ষর ল্যাবস' (Akkhar-Labs)-এর সার্বভৌম কম্পিউটিং স্ট্যাক তৈরির উদ্যোগের প্রথম ও প্রধান গাণিতিক সোপান। প্রচলিত পশ্চিমা কম্পিউটিং কাঠামোর ওপর নির্ভরতা কমিয়ে একটি সম্পূর্ণ দেশীয় ও স্বকীয় লজিক্যাল ইকোসিস্টেম গড়ে তোলাই এই প্রকল্পের মূল লক্ষ্য।

## স্থাপত্যের রূপরেখা (Architecture Overview)

বয়ন অ্যাসেম্বলারটি **নির্দেশিকা-১৬** নামক একটি ১৬-বিট যন্ত্র-বিধানের (ISA) ওপর ভিত্তি করে নির্মিত। এর প্রধান কারিগরি বৈশিষ্ট্যসমূহ:

- **আর্কিটেকচার:** ১৬-বিট নেটিভ।
- **বিট বিন্যাস (Endianness):** লিটল-এন্ডিয়ান (Little-endian)।
- **নির্দেশ বিন্যাস (4-4-8 Format):** 
    - ওকোড (Opcode): ৪-বিট
    - সাময়িক আইডি (Register ID): ৪-বিট
    - মান/ঠিকানা (Immediate/Address): ৮-বিট
- **এনকোডিং মানদণ্ড:** অক্ষর-লিপি ৮ (Akshar-Lipi 8)।
- **আউটপুট ফরম্যাট:** `.ব` (সার্বভৌম র-বাইনারি ফাইল)।

## মূল বৈশিষ্ট্যসমূহ

### ১. মৌলিক ওকোড ম্যাপিং
বয়ন সরাসরি বাংলা সিনট্যাক্স প্রসেস করে। এখানে `LOAD` বা `MOV` এর পরিবর্তে `আদান` বা `স্থানান্তর`-এর মতো মৌলিক বাংলা ওকোড ব্যবহৃত হয়, যা হার্ডওয়্যার লেভেলে সরাসরি গাণিতিক মানের সাথে ম্যাপ করা।

### ২. অক্ষর-হেক্স (Akkhar-Hex) পদ্ধতি
ঐতিহ্যবাহী ষোড়শিক (Hexadecimal) পদ্ধতির পরিবর্তে এখানে একটি স্বকীয় পদ্ধতি অনুসরণ করা হয়েছে। ০-৯ সংখ্যার পাশাপাশি ১০-১৫ পর্যন্ত মানের জন্য বাংলা বর্ণমালার প্রথম ৬টি ব্যঞ্জনবর্ণ (ক, খ, গ, ঘ, ঙ, চ) ব্যবহার করা হয়েছে।

### ৩. দেশীয় সংখ্যা পদ্ধতি
এটি সরাসরি বাংলা সংখ্যা (০-৯) শনাক্ত করতে এবং সেটিকে গাণিতিক উপাত্তে রূপান্তর করতে সক্ষম। এটি কেবল একটি অনুবাদক নয়, বরং এটি মেমোরি লেভেলে বাংলাকে একটি গাণিতিক সত্তা হিসেবে প্রতিষ্ঠা করে।

## প্রোজেক্ট কাঠামো

- `include/arch/`: নির্দেশিকা-১৬ এর হার্ডওয়্যার ডেফিনিশন।
- `src/lexer.c`: বাংলা শব্দ ও সংখ্যা শনাক্তকারী লেক্সিক্যাল ইঞ্জিন।
- `src/parser.c`: ৪-৪-৮ বিন্যাসে বাইনারি প্যাক করার সিনট্যাক্স ইঞ্জিন।
- `src/main.c`: সোর্স ফাইল রিডার ও বিল্ড ম্যানেজমেন্ট।

## ব্যবহার নির্দেশিকা

### বিল্ড করার পদ্ধতি (Windows/Linux)
প্রকল্পটি কম্পাইল করার জন্য আপনার সিস্টেমে GCC থাকা আবশ্যক। টার্মিনালে নিচের কমান্ডটি ব্যবহার করুন:

```bash
gcc -Iinclude src/*.c -o bin/boyon
```

### অ্যাসেম্বল করার পদ্ধতি
একটি বাংলা অ্যাসেম্বলি সোর্স কোডকে বাইনারিতে রূপান্তর করতে:

```bash
./bin/boyon
```
এটি সফলভাবে আপনার সোর্স কোডকে `নির্দেশিকা.ব` নামক র-বাইনারি ফাইলে রূপান্তর করবে, যা পরবর্তীতে নির্দেশিকা-১৬ সমর্থিত সিমুলেটর বা হার্ডওয়্যারে সরাসরি রান করা সম্ভব।
